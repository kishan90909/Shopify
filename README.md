## 🚀 Overview

A modern and responsive bookstore landing page inspired by Shopify storefront designs. The project focuses on clean UI, structured layouts, promotional sections, and a seamless browsing experience using pure HTML and CSS.

---

## ✨ Features

* Responsive Design
* Sticky Navigation Bar
* Hero Banner Section
* Featured Books Showcase
* Author Spotlight Section
* Promotional Banners
* Customer Reviews
* Newsletter Subscription
* Modern UI & Hover Effects

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Font Awesome
